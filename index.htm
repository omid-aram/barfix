Barfix Layout & Grid System

Use this simple mobile-first responsive grid to build layouts of your web pages. It's not just a twelve column system, but a 1..10, 12, 16 and 25 column system!

How it works

Barfix's grid system is so simple and easy to learn. It's fully responsive using CSS float. There is a "bx-float" class for any container and the rest is all about column widths.

div class="bx-float" or bx-float
.bx-1-9
.bx-sm-1-7
.bx-md-2-10
.bx-lg-2-12
.bx-xl-3-16
.bx-xxl-3-25

Hint: "bx-sm-2-5" sets width to 2/5 of the screen when it is "small" or greater.

Also we have a "bx-table" class for any container to behave its children as table-cell. Size the cells by setting "bx-w-1" to "bx-w-25"; The number (1..25) indicates weight of each cell rather than others.

div class="bx-table" or bx-table
.bx-w-1
.bx-w-2
.bx-w-3

Place it, where ever you want

There is also a positioning classes for nine positions of a box. Just be carefull about container's width, height and position.

.bx-top-left
.bx-top-center
.bx-top-right
.bx-middle-start
.bx-middle-center
.bx-middle-end
.bx-bottom-left
.bx-bottom-center
.bx-bottom-right

Shape it, how ever you want

Preferably for images, it has a ratio system to set the image (or any content) size to a specific aspect ratio. Developed for image layout pages like Instagram.

.bx-ratio-1-1 (Using Image)
.bx-ratio-3-4 (Using Background)
.bx-ratio-9-16
.bx-ratio-16-9
