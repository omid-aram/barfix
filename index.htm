Barfix Layout & Grid System

Use this simple, mobile-first, responsive grid to build the layouts of your web pages. It’s not just a twelve-column system — it supports 1 to 10, 12, 16, and 25 column grids!

How it works

The Barfix grid system is simple and easy to learn. It is fully responsive using CSS floats. Any container with the class ".bx-float" works as a floatable container, and you set the widths of child elements by applying the appropriate column width classes.

div class="bx-float" or bx-float
.bx-1-9
.bx-sm-1-7
.bx-md-2-10
.bx-lg-2-12
.bx-xl-3-16
.bx-xxl-3-25

Hint: The class ".bx-sm-2-5" sets the width of an element to 2/5 of the container when the viewport is at least the "small" breakpoint.

Also we have a "bx-table" class for any container to behave its children as table-cell. Size the cells by setting "bx-w-1" to "bx-w-25"; The number (1..25) indicates weight of each cell rather than others.

div class="bx-table" or bx-table
.bx-w-1
.bx-w-2
.bx-w-3

Shape it, how ever you want

The ratio system allows you to set the size of images (or any content) to specific aspect ratios. This is especially useful for image layout pages, like Instagram.

.bx-ratio-1-1 (Using Image)
.bx-ratio-3-4 (Using Background)

Inspired by Tailwind color palette, we have 12 main colors, with 10 shades; plus Black and White (with no shade).

How to use

div class="bx-{color}-{shade}"

Shade No.5 is default in light mode, and it's No.4 in dark mode.

Such as: div class="bx-red-5"
Equal to: div class="bx-red"

Text color

You can set all colors as text color of an element:

span class="bx-text-{color}" or {color}

Such as: span class="bx-text-pink" It's pink/span
Equal to: pink It's pink/pink

Border color?

Yes, just set "bx-border" class or use bx-border tag to set that for all inner elements:

Such as: div class="bx-lime-2 bx-border"
Or: bx-border...div class="bx-lime-2" .../bx-border

Hover color?

Yes, just set "bx-hover" class or use bx-hover tag to set that for all inner elements:

Such as: div class="bx-cyan-6 bx-hover"
Or: bx-hover...div class="bx-cyan-6" .../bx-hover

Dark mode?

Yes, just set "bx-dark" class to an outer element:

Such as: body class="bx-dark"
